# Image URL Extraction Guide

Since automated downloads are having issues, here's a manual process to get image URLs:

## Quick Method

1. Visit the weapon's wiki page (e.g., `https://escapefromtarkov.fandom.com/wiki/AK-74M`)
2. Right-click the main weapon image
3. Select "Copy image address"
4. You'll get a URL like: `https://static.wikia.nocookie.net/escapefromtarkov_gamepedia/images/a/a1/Ak74m.png/revision/latest/scale-to-width-down/320?cb=20231017095858`
5. Extract just the path part: `a/a1/Ak74m.png`
6. Add it to the script's hashtable
